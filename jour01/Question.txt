🔧 Job01 : Qu'est-ce que Spring Initializr et comment peut-il faciliter la création d'un nouveau projet Spring?

    Spring Initializr est un outils qui permet de créer simplement un projet Spring Boot avec les dépendances nécessaires.
    Il simplifie la configuration initiale en générant un projet prêt à être importé.


📄 Job02 : Pourquoi le fichier pom.xml est-il crucial dans un projet Maven ?

    Le fichier pom.xml est un fichier de configuration essentiel pour un projet Spring Boot utilisant Maven.
    Il contient les informations du projet, les dépendances, les plugins et d'autres configurations nécessaires à la gestion de la construction du projet.


🖥️ Job03 : Qu'est-ce qu'un contrôleur dans le contexte de Spring MVC ?

    Les contrôleurs sont des classes Java chargées de gérer les requêtes HTTP entrantes et de renvoyer une réponse appropriée .
    Ils servent d'intermédiaires entre le client (généralement un navigateur web ou une application mobile) et la logique métier de l'application.


📄 Job04 : Comment Spring permet-il l'injection de propriétés depuis des fichiers de configuration ?

    Spring permet l'injection de propriétés depuis application.yml grâce à @Value pour une injection directe.
    Ces fichiers sont automatiquement chargés au démarrage de l'application.


🖥️ Job05 : Pourquoi serait-il utile d'avoir différents profils dans une application Spring ?

    Les profils permettent de séparer la configuration de l'application selon l'environnement dans lequel elle fonctionne.
    Cela permetterait de pouvoir avoir un environnement de développement et un environnement de production indépendant qui
    peuvent avoir des fonctionnalités différentes.

🔧 Job06 : En quoi la dépendance DevTools est-elle bénéfique pour le développement ?

    Le DevTools facilite le développement grâce au redémarrage automatique de l'application, à l'affichage amélioré des erreurs
    ce qui permet de visualiser directement les modifications apportées au code.
